A boilerplate generator for an ExpressJS with Typescript, with QOLs such as ESLint, Prettier and Husky ✨✨✨

## Installation :zap:

Use an NPX script to get everything setup and ready to use:

```bash
npx @dtrn/express-ts-template PROJECT_NAME
```
